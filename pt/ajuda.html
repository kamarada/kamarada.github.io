Ajuda

Se você precisa de mais informações ou está com alguma dúvida sobre Linux, experimente as opções listadas nesta página.

  • Ajuda específica para o Linux Kamarada

    Se você usa a distribuição Linux Kamarada, o projeto disponibiliza as seguintes opções de suporte:
    • Pesquisa no site e na Internet
      Este site é a principal fonte de informação sobre o Linux Kamarada. Use o campo Pesquisar na barra de navegação no topo. Você também pode fazer uma busca na Internet usando seu mecanismo de busca preferido.
    • Lista de discussão por e-mail / grupo do Google
      Você pode lançar sua dúvida no grupo do Google do Linux Kamarada, que pode ser usado semelhante a uma lista de discussão por e-mail ou diretamente a partir do navegador.
    • Redes sociais
      Siga o Linux Kamarada nas redes sociais da seção Sobre desta página. Você pode enviar mensagens ou deixar comentários, que serão respondidos pelo projeto ou mesmo por outros seguidores.

  • Ajuda para o Linux openSUSE

    O Linux Kamarada é baseado na distribuição openSUSE. Dúvidas enviadas para o Linux Kamarada também podem ser enviadas para o openSUSE (nesse caso, lembre-se de informar que usa uma distribuição baseada no openSUSE).
    • Guia do openSUSE
      Comece por aqui. Esse guia contém tudo que os novatos precisam saber para começar a usar o openSUSE. A tradução para o Português Brasileiro é feita pelo Linux Kamarada e está em andamento.
    • Grupo do Telegram
      Se você usa o aplicativo de mensagens Telegram, entre no grupo @opensusebr, que é bastante ativo. Lá você pode lançar sua pergunta, que será respondida por usuários ávidos para ajudar.
    • Listas de discussão por e-mail
      As listas de discussão por e-mail do openSUSE também são bastante movimentadas. Os usuários fazem e respondem perguntas e é possível contatar os mantenedores do projeto.
    • Fóruns do openSUSE
      Alguns usuários acompanham mais as listas de discussão, outros preferem os fóruns, que são outra opção para obter suporte e trocar ideias.
    • Outras opções de ajuda e documentação
      No Apêndice A do Guia do openSUSE são listados vários lugares onde você pode ler mais sobre o openSUSE, assim como obter ajuda de usuários online.

  • Ajuda para Linux em geral

    Neste site você consegue informações e ajuda para distribuições Linux em geral (incluindo o Linux Kamarada e o openSUSE):
    • Viva o Linux
      Contém artigos, dicas e scripts contribuídos pela comunidade e possui um fórum para dúvidas. É provavelmente a fonte de informação mais completa sobre Linux em Português.

Sobre

O Projeto Linux Kamarada visa divulgar e promover o Linux como um sistema operacional robusto, seguro, versátil e fácil de usar, adequado para o uso diário seja em casa, no trabalho ou no servidor. Os focos do projeto são principalmente distribuição e documentação.
